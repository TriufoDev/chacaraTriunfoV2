# Chácara Triunfo - Landing Page

Este projeto é uma Landing Page responsiva e otimizada para o aluguel de temporada da **Chácara Triunfo**. O site foi desenvolvido com foco em conversão, velocidade e experiência do usuário (UX).

## 🚀 Tecnologias Utilizadas

*   **HTML5 Semântico**: Estrutura otimizada para SEO.
*   **Tailwind CSS (CDN)**: Framework CSS utilitário para estilização rápida e responsiva.
*   **JavaScript (Vanilla)**: Lógica para menu mobile, loader e integração com WhatsApp.
*   **Swiper.js**: Biblioteca para o carrossel de fotos (touch-friendly).
*   **GLightbox**: Biblioteca para visualização de galeria em tela cheia (Lightbox).
*   **Font Awesome**: Ícones vetoriais.
*   **Google Fonts**: Tipografia (Poppins).

## 📂 Estrutura de Arquivos

- `index.html`: Página principal (Single Page Application style).
- `termos.html`: Página de Termos e Condições de uso.
- `script.js`: Lógica de interação, animações e integrações.

## ✨ Funcionalidades

1.  **Responsividade Total**: Layout adaptável para Celulares, Tablets e Desktops.
2.  **SEO Avançado**: Meta tags, Open Graph, Schema.org (JSON-LD) e Canonical tags.
3.  **Galeria Interativa**: Carrossel com zoom em imagens que se adaptam à tela.
4.  **Reserva via WhatsApp**: O formulário de contato gera uma mensagem automática formatada para o WhatsApp do proprietário.
5.  **Performance**: Carregamento lazy (preguiçoso) de imagens e fontes.

## 🛠️ Como Executar

Como o projeto utiliza bibliotecas via CDN, não é necessário instalação de pacotes (npm/yarn).

1.  Baixe os arquivos.
2.  Abra o arquivo `index.html` em qualquer navegador moderno.
